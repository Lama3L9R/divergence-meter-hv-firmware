DM-HV Module can boost 5V DC input into 154.929V DC - 177.525V DC depend on the switch configuration.

| Input                     | Output Min    | Output Max    | Max Current | 
|---------------------------|---------------|---------------|-------------|
| `5V DC` And both `3V3 DC` | `154.929V DC` | `177.525V DC` | `350 mA`    |
